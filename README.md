# MinGym

A sleek, minimalist fitness tracking application built to help users log workouts, manage custom routines, and track progressive overload with analytics.

## Features
* **Offline-First Architecture**: Log workouts in the gym basement. Data caches locally and syncs when the network returns.
* **Live Workout Tracking**: Manage active sets, weight, and reps with an integrated, background rest timer.
* **Analytics Tracking**: Interactive charts for volume, duration, and 1RM (One Rep Max) estimates.
* **Custom Routines & Exercises**: Build personalised workout templates and add custom movements.
* **Secure Authentication**: Anonymous guest access that seamlessly upgrades to secure Email/Password accounts.
* **Data Ownership**: CSV export of all historical workout data.

## Tech Stack
* **Framework**: Flutter
* **Backend**: Firebase (Auth, Cloud Firestore)
* **Architecture**: Stream-based reactive UI
